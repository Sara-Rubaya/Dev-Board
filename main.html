Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in the DOM are: By 'ID',by 'Class Name' , by 'Tag Name',by 'CSS Selector' ,by 'Name Attribute' .

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML : Returns or sets the HTML content of an element, including tags and nested elements.
innerText : Returns or sets only the visible text inside an element, ignoring hidden elements and extra spaces.
textContent : Returns or sets all text inside an element, including hidden text, and preserves spacing.

Question-3: What is event delegation in the DOM?

Event delegation means adding a single event listener to a parent element to handle events for its child elements using event bubbling. It keeps code simple and efficient!

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts on an element and "bubbles up" to its parents, letting them react to it too.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create, add, and remove elements using JavaScript are:
Create an element: document.createElement("tag")
Add it to the DOM: parent.appendChild(element)
Remove an element: element.remove()
